Change the hostname:

1. sudo nano /etc/hostname ---Change the to

2. sudo nano /etc/hosts ---Change the to

3. sudo service hostname restart

Change the username:

1. Ctrl+Alt+F1

2. log in using your former username and password

3. sudo passwd root ---set a password for the "root" account

4. exit ---log out

5. log in using "root" and the new password

6. usermod -l -d /home/ -m

7. exit ---log out

8. Ctrl+Alt+F7
